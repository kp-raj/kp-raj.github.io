Impact Hub

Tech stack used: Django, Python, Sqlite3, HTML/CSS, Bootstrap, JavaScript, jQuery

Deloitte conducts an impact day every year where every employee must participate in an event for giving back to the community.

I built a web application on Django framework, as a firm initiative, for employees to choose between various events at various places. It has features from signing up, logging in, submitting the choices to sending the confirmation mails of assigned events.

Due to the current pandemic, impact day services have been shifted to online video calls. So, I built an additional extension for an admin to be able to generate and send zoom meeting links to attendees of each event automatically, using Zoom API and JWT.
